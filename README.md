# Bank Marketing Analysis

Analyse du dataset Bank Marketing de l'UCI.
Dataset : 45 211 clients d'une banque portugaise (2008-2010).

## Objectif
Analyser les campagnes de marketing téléphonique pour comprendre
quels clients souscrivent à un dépôt à terme.

## Outils utilisés
- Python (pandas, matplotlib)
- SQL (SQLite)
- Power BI
- Excel

## Fichiers
- bank_python.ipynb : analyse exploratoire en Python
- bank_sql.ipynb : analyse SQL avec SQLite
- bank_debutant.pbix : dashboard Power BI
- bank_analyse.xlsx : analyse Excel

## Résultats clés
- Taux de souscription global : 11.7%
- Meilleur mois : mars (52%)
- Meilleur métier : étudiant (28.7%)
- Les clients qui souscrivent ont un solde moyen plus élevé (1804€ vs 1304€)
